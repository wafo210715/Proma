# Proma v0.15.1 更新

## Bug 修复

- **修复 Pi 写文件工具参数不完整问题** — 明确要求 `write` 在同一次调用中提供路径和完整 `content`，避免仅传路径导致写入失败。(#1184)
- **优化 Pi 工具调用流式预览** — 预览阶段不再重复传输累积的工具参数，降低大文件写入时的 IPC 与渲染压力。(#1185)
