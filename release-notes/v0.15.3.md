# Proma v0.15.3 更新

## 新功能

- **ChatGPT Codex 快速模式** — Pi Runtime 下使用受支持 Codex 模型时，可在会话工具栏切换「快速」模式，以优先响应换取更高的额度消耗；该设置会随会话保存。

## Bug 修复

- **优化 Pi 流式输出性能** — 将高频、累计式的 partial 消息合并为最高 20 fps 的最新快照，同时确保最终内容立即送达，减少长回复时的 IPC 与渲染压力。
